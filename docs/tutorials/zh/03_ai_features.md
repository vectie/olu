# AI 功能使用指南 (AI Features)

利用内置的生成式 AI，您可以大幅提升建模效率。

## 1. AI 助手 (AI Assistant)
点击右上角的 **"AI 助手"**。
- **自然语言建模**：您可以输入 "创建一个四足机器人的腿部结构"。
- **批量修改**：尝试输入 "将左腿所有的颜色改为红色"。
- **硬件建议**：输入 "推荐适合这个膝关节的电机"。

## 2. AI 审阅 (AI Inspector)
点击 **"AI 审阅"** 按钮进行质量审计。
- **评分体系**：每个检查项 0-10 分。
- **分类维度**：
    - **物理合理性**：质量是否为正？惯性矩阵是否满足三角不等式？
    - **坐标系对齐**：关节轴是否统一？
    - **命名规范**：是否存在重名？
- **导出报告**：支持将审阅结果导出为 PDF 文档。
