# 设计模式详解 (Design Modes)

URDF Studio 提供三种专业模式，覆盖机器人设计的不同阶段。

## 1. 骨架模式 (Skeleton Mode)
**目标**：确立机器人的拓扑结构。
- **添加 Link**：点击树状图中的 `+` 按钮。
- **调整关节 (Joint)**：
    - 移动关节位置会改变子连杆相对于父连杆的偏移。
    - 在属性面板修改关节类型（Fixed, Revolute, Continuous, Prismatic）。

## 2. 细节模式 (Detail Mode)
**目标**：定义机器人的物理外形。
- **视觉模型 (Visual)**：设置颜色或上传 STL/OBJ/DAE 网格。
- **碰撞模型 (Collision)**：
    - **性能优化**：建议使用基础几何体（Box, Cylinder）作为碰撞体，而非直接复制视觉网格。
    - **实时加载**：修改碰撞体参数时，3D 视图会局部刷新，提供即时反馈。
- **惯性参数**：设置质量 (Mass) 和惯性矩阵 (Inertia)。

## 3. 硬件模式 (Hardware Mode)
**目标**：配置机电规格。
- **电机库选型**：从内置的 Unitree、RobStride 等品牌中选择电机。
- **自动同步**：系统将自动填充力矩限制 (Effort) 和速度限制 (Velocity)。
- **限位设置**：根据机械结构调整关节的转动角度范围。
