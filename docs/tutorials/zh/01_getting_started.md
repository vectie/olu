# 快速入门指南 (Getting Started)

欢迎使用 **URDF Studio**！本指南将带您快速了解如何从零开始设计您的第一个机器人。

## 1. 核心流程预览
设计一个机器人通常遵循以下顺序：
1. **构建骨架**：在 `Skeleton` 模式下定义层级结构。
2. **细化几何**：在 `Detail` 模式下添加视觉模型和碰撞体。
3. **配置硬件**：在 `Hardware` 模式下选型电机。
4. **AI 审阅**：检查物理合理性。
5. **导出**：生成仿真包。

## 2. 界面布局
- **左侧面板**：机器人树 (Robot Tree)，管理 Link 和 Joint 的父子关系。
- **中间区域**：3D 视图窗口，支持 Gizmo 实时变换操作。
- **右侧面板**：属性编辑器 (Property Editor)，调节具体的数值参数。
- **顶部栏**：切换模式、AI 工具及导入导出。

## 3. 基本操作
- **平移/旋转**：点击 3D 视图中的组件，使用出现的箭头/圆环进行拖拽。
- **聚焦**：点击树状图中的节点，视图将自动对焦到该部件。
- **撤销/重做**：使用键盘快捷键 `Ctrl+Z` / `Ctrl+Y`。
